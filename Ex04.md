# Ex. No : 4	
# RECOGNITION OF A VALID VARIABLE WHICH STARTS WITH A LETTER FOLLOWED BY ANY NUMBER OF LETTERS OR DIGITS USING YACC
## Register Number : 212224040083
## Date : 26-05-2026

## AIM   
To write a YACC program to recognize a valid variable which starts with a letter followed by any number of letters or digits.

## ALGORITHM
1.	Start the program.
2.	Write a program in the vi editor and save it with .l extension.
3.	In the lex program, write the translation rules for the keywords int, float and double and for the identifier.
4.	Write a program in the vi editor and save it with .y extension.
5.	Compile the lex program with lex compiler to produce output file as lex.yy.c. eg $ lex filename.l
6.	Compile the yacc program with YACC compiler to produce output file as y.tab.c. eg $ yacc –d arith_id.y
7.	Compile these with the C compiler as gcc lex.yy.c y.tab.c
8.	Enter a statement as input and the valid variables are identified as output.

## PROGRAM

#### variable_test.l

```c
%{
#include "variable_test.tab.h"
#include <stdio.h>
%}

%%

"int"       { return INT; }
"float"     { return FLOAT; }
"double"    { return DOUBLE; }

[a-zA-Z][a-zA-Z0-9]* {
    printf("\nIdentifier is %s", yytext);
    return ID;
}

[ \t]       ;

\n          { return '\n'; }

.           { return yytext[0]; }

%%

int yywrap()
{
    return 1;
}
```


#### variable_test.y

```c
%{
#include <stdio.h>
#include <stdlib.h>

void yyerror(char *s);
int yylex(void);
%}

%token ID INT FLOAT DOUBLE

%%

D : T L ';'
    {
        printf("\nValid Variable Declaration\n");
    }
    ;

L : L ',' ID
  | ID
  ;

T : INT
  | FLOAT
  | DOUBLE
  ;

%%

int main()
{
    printf("Enter declaration:\n");
    yyparse();
    return 0;
}

void yyerror(char *s)
{
    printf("\nInvalid Variable Declaration\n");
}
```
## Execution Steps

- Generate Bison files

  ```bash
  bison -d variable_test.y
  ```

- Generate Flex file

  ```bash
  flex variable_test.l
  ```

- Compile

  ```bash
  gcc lex.yy.c variable_test.tab.c -lfl
  ```

- Run

  ```bash
  ./a.out
  ```
